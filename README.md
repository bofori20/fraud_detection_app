# Fraud Detection App 🛡️

This project uses machine learning to detect fraudulent transactions.

## 📊 Dataset Notice
Due to GitHub's file size limits, the **AIML Dataset.csv** (470MB) is not included in this repository. 

**To run this project locally:**
1. Download the dataset from https://www.kaggle.com/datasets/amanalisiddiqui/fraud-detection-dataset.
2. Place the file in the root directory of this project.
3. Ensure the filename matches `AIML Dataset.csv`.
